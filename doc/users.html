Users endpoint

User JSON representation


Create a user

POST /users

Description

Creates a new user and returns the name.


Responses

  • 200 OK
    JSON example: 
    { "name" : "test" }
  • 400 Bad Request
  • 409 Username taken

Update a user

Put /users

Description

Change the name and password of the authenticated user. Returns the new name.,


Responses

  • 200 OK
    JSON example: 
    { "name" : "test2" }
  • 400 Bad Request
  • 401 Unauthorized
  • 409 Username taken
