Character JSON representation

All fields are required for the json to be accepted.


Character fields

Field Name Type Description
name string The name of the character
background string The character's background
race string The character's race
str integer The character's strength
dex integer The character's dexterity
con integer The character's constitution
intel integer The character's intelligence
wis integer The character's wisdom
cha integer The character's charisma
ac integer The character's armor class
init integer The character's initiative bonus
speed integer The character's movement speed in feet
maxHp integer The character's maximum hit points
skills list<string> The skill proficiencies of the character
languages list<string> The languages the character knows
tools list<string> The tool proficiencies of the character
items Map<string,integer> The names and quantities of tools the character has
features list<string> The features of the character
classes list<class> The classes of the character

Class fields

Field Name Type Description
level integer The amount of levels in the class
name string The name of the class

Example:

{ "name" : "JEB", "background" : "Thief", "race" : "Elf", "str" : 10, "dex" : 11, "con" : 12, "intel" : 14, "wis" : 13, "cha" : 15, "ac" : 14, "init" : 0, "speed" : 30, "maxHp" : 27, "skills" : [ "Stealth", "Persuasion", "Sleight of Hand" ], "languages" : [ "Common", "Elvish" ], "tools" : [ "Thieves' tools" ], "items" : { "Spoon" : 176, "Dagger" : 5 }, "features" : [ "Sneak Attack", "Disengage" ], "classes" : [ { "level" : 1, "name" : "rogue" }, { "level" : 1, "name" : "wizard" } ], "id" : 12 }
