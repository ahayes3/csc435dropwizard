Characters endpoint

User JSON representation


List owned characters

GET /characters

Description

Returns a list of the ids of all characters owned by the authenticated user.


Responses

  • 200 OK
    JSON example: 
    [ 12, 13 ]
  • 401 Unauthorized
  • 403 Forbidden
